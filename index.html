Project Overview

This website is a simple Numerical Analysis project that calculates roots of equations using different numerical methods.

Each method has its own page with inputs, a calculation button, a final result, and an iteration table or output area.

How to Use the Calculator

First, choose one of the methods below. Then enter the polynomial terms, starting values, tolerance, and maximum number of iterations.

After clicking the solve button, the page will show the approximate root and the steps used to reach it.

Available Root-Finding Methods

Bisection Method

Uses an interval [a, b] and repeatedly cuts it in half until the root is found.

Open Bisection

False Position Method

Uses two points around the root and finds a better estimate using a straight line.

Open False Position

Secant Method

Uses two starting guesses and approximates the root without needing the derivative.

Open Secant

Newton-Raphson Method

Uses one starting guess and the derivative of the function to approach the root.

Open Newton-Raphson
