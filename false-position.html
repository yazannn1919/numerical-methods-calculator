False Position Method

The False Position Method is used to find the root of a function inside an interval.

It starts with two points x(n-1) and x(n). The method requires f(x(n-1)) and f(x(n)) to have opposite signs.

Formula:
x(n+1) = x(n) - f(x(n)) × (x(n) - x(n-1)) / (f(x(n)) - f(x(n-1)))
