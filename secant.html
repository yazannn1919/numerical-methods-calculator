Secant Method

The Secant Method is used to find the root of a function using two starting values.

It is similar to Newton-Raphson, but it does not need the actual derivative. Instead, it uses two previous points to estimate the next value.

Formula: x(n+1) = x(n) - (f(x(n)) × (x(n) - x(n-1))) / (f(x(n)) - f(x(n-1)))
